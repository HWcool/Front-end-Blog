## 安装
1. 安装MongoDB并成功启动
2. git clone "https://github.com/luosijie/card](https://github.com/luosijie/card.git"
3. npm install
4. npm start
5. 访问 localhost:3000

## 实现功能
1. 登录注册
2. 图片简单处理
3. 名片模板的上传和保存
4. 名片图片的打包和下载
4. 名片模板收藏
5. MongoDB数据增删改查

## 使用到的插件
1. html2Canvas 用于将html解析为图片
2. jsZip 用于图片打包
3. fileSaver 用于导出打包后的zip文件
